# **App Name**: TripWise Navigator

## Core Features:

- AI Itinerary Generation: Generate personalized trip itineraries based on destination, duration, budget, and travel themes, using the Gemini API.
- Dynamic Itinerary Enrichment: Enrich the itineraries with real-time data like location coordinates, photos, and travel times using Google Maps API.
- Real-Time Adjustment Tool: Modify itineraries based on current conditions (e.g., weather) by using the AI model as a tool.
- Itinerary Display: Display itineraries in a clear, mobile-first interface using cards or lists.
- Mock Booking Simulation: Simulate booking flows with dummy payment processing. Transactions will always be marked as successful.
- Itinerary Export and Sharing: Enable users to export itineraries as PDFs and share them with others.
- User Authentication: Implement user authentication using Firebase Authentication with Google Sign-In.

## Style Guidelines:

- Primary color: Light indigo (#7986CB) for a calm, travel-oriented feel.
- Background color: Very light indigo (#F2F3F8), offering a gentle contrast and spacious feel.
- Accent color: Muted violet (#9FA8DA) to highlight interactive elements.
- Body and headline font: 'PT Sans' (sans-serif) for a modern and accessible interface. 
- Note: currently only Google Fonts are supported.
- Use consistent and clear icons to represent different itinerary elements and actions.
- Mobile-first design with a clean, card-based layout for itineraries.
- Use subtle animations to enhance user interactions, like itinerary loading and real-time updates.
